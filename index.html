Learn Component Library

Ready-made HTML components designed for XAMK Learn pages. Pick any component, fill in the content, and paste the generated HTML directly into Moodle's HTML source editor.

1

Pick a component

Browse the sidebar on the left. Click any component to load a live preview. Star your favourites to pin them to the top.

2

Edit the content

Use the config panel on the left to fill in your text, dates, and options. The preview updates instantly as you type.

3

Copy and paste into Moodle

Click Copy HTML in the top bar. In Moodle, open the page editor, switch to HTML source view, and paste.

All components use inline styles only — no external CSS needed.
They render correctly inside Moodle without any extra setup.
